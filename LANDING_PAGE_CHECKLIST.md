# 랜딩페이지 체크리스트 평가 문서 (v2.1)

## 평가 대상
- **서비스**: 안전의 길 - 게이미피케이션 안전관리 플랫폼
- **평가 일자**: 2024-12-16
- **평가 버전**: v2.1 (게이미피케이션 + 자발적 참여 중심)

---

## STEP 0. 내 서비스 유형 진단하기

- [ ] **A 유형 (불안 해소형):** 전문가용 서비스, 논문/법률/의료 등 '실수'가 없어야 하는 분야
- [ ] **B 유형 (기술 몰입형):** 개발자/디자이너 등 전문가가 직접 쓰는 도구
- [x] **C 유형 (결과 지향형):** 복잡한 과정을 줄여주는 자동화 솔루션

> **선택 이유**: 복잡한 안전관리 업무(점검·승인·조치)를 **게이미피케이션**으로 **자발적 참여**를 유도하고, 워크플로우 자동화 + AI 위험 판별로 "입력하면 증빙이 나온다"는 결과 지향적 가치를 제공

## STEP 1. [공통] 핵심 코어 필수요소 점검 (6/6 ✅)

### ① 히어로 섹션 (Hero Section) - 첫인상 3초

- [x] **명확한 헤드라인:** 고객이 얻을 '최종 이득'이 한 문장으로 정리되었는가?
  > ✅ **"중대재해를 막는 안전관리, 이제 '간접 실행'에서 '직접 실행'으로"**
  > 안전관리의 패러다임 전환(수동적 관리 → 자발적 참여)을 명확히 제시

- [x] **보조 설명 (Sub-head):** 헤드라인의 이득을 얻는 구체적인 방법/기술이 설명되었는가?
  > ✅ "게이미피케이션을 적용한 안전관리 솔루션 - 현장 안전관리를 자발적으로 모두가 참여하게 하는 플랫폼"
  > 방법론(게이미피케이션, 자발적 참여)을 구체적으로 제시

- [x] **직관적 이미지:** 서비스의 정체성을 한눈에 보여주는 시각 자료가 있는가?
  > ✅ 다크모드 UI + 그라데이션 배경 + 신뢰 지표 수치로 전문적인 B2B SaaS 인상 전달

### ② 행동 유도 (CTA) - 전환 장치

- [x] **동사형 버튼:** 행동 유도형 문구인가?
  > ✅ "무료로 시작하기", "데모 영상 보기", "무료 체험" - 모두 행동 유도형

- [x] **반복 배치:** 상단, 중간, 하단에 버튼이 노출되는가?
  > ✅ Hero 섹션(2개), Final CTA 섹션, 플로팅 버튼 - 4곳 배치

### ③ 신뢰의 증거 (Social Proof) - 의심 제거

- [x] **숫자/로고 증명:** 객관적 지표가 있는가?
  > ✅ "100% 법정 점검표 기반", "3단계 워크플로우 자동화", "실시간 AI 위험도 분석"

## STEP 2. [선택] C유형 전략 요소 점검 (3/3 ✅)

### 🅲 C 유형 (Macaly 전략) - 핵심 목표: "편의성"과 "결과물"

- [x] **[Input-Output]** 복잡한 과정은 숨기고, "입력하면 결과가 나온다"는 단순한 구조로 표현했는가?
  > ✅ **역할 기반 워크플로우 자동화**
  > - 기술인(체크리스트 작성·사진 업로드) → 관리감독자(위험 항목 중심 승인/반려) → 안전관리자(전체 현장 위험현황 조회 및 조치 기록)
  > - "점검→승인→조치 전 과정 자동 기록 → 감사·사고·분쟁 시 객관적 근거 확보"

- [x] **[결과 갤러리]** 이 툴로 만들어진 완성도 높은 결과물 예시를 보여주었는가?
  > ✅ **AI 안전 지능 시각화**
  > - 안전(✅) / 주의(⚠️) / 위험(🚨) 3단계 판별 결과
  > - 위험 태그 자동 제공 (보호구 미착용, 추락 위험 등)
  > - 고위험 작업 상위 노출

- [x] **[Before & After]** 도입 전후의 효과를 비교했는가?
  > ✅ **핵심 문제 섹션** (Before)
  > - 법정 자율점검·안전서류 수기·분산 관리로 누락/중복 발생
  > - 작업자-감독자-안전관리자 간 워크플로우 단절
  > - 사진은 있지만 객관적 위험 판단 기준 없음
  > - 사고 발생 시 책임 추적 부담 (점검/승인/조치 여부 입증 어려움)
  > 
  > ✅ **핵심 솔루션 섹션** (After)
  > - 법정 기준 체크리스트 전산화 (사다리/고소작업대/밀폐공간 표준 템플릿)
  > - 역할 기반 워크플로우 자동화 (점검→승인→조치 전 과정 자동 기록)
  > - AI 기반 사진 위험도 분석 (안전/주의/위험 자동 판별)
  > - "내 할 일" 중심 실무 UX (역할별 우선순위 업무만 노출)

- [x] **안전관리 책임 추적:** 모든 점검·승인·조치 타임스탬프 기록
- [x] **법적 리스크 최소화:** 중대재해처벌법 대응 객관적 근거 확보
- [x] **감사 대응:** 즉시 제출 가능한 데이터

### 핵심 기능 체크리스트
- [x] **체크리스트 시스템:** 법정 자율점검표 기반 표준 템플릿, 모바일 체크리스트 작성 및 제출, 자동 취합 및 상태 관리
- [x] **워크플로우 자동화:** 역할별(기술인/관리감독자/안전관리자) 업무 분리 및 자동 연계
- [x] **AI 안전 지능:** 작업사진 위험도 분석, 위험 태그 자동 제공, 고위험 작업 상위 노출
- [x] **증빙 체계:** 타임스탬프 기록, 감사·점검 시 즉시 제출 가능, 중대재해처벌법 대응 근거 확보

- [x] **차별화 키워드:** "AI 위험도 분석", "작업사진 위험 판별", "예방 중심 안전관리", "데이터 기반 안전관리"
- [x] **캐치프레이즈:** "중대재해를 막는 안전관리, 이제 '증명'까지 자동으로", "점검은 했는데, 증명이 안 되는 문제를 끝내다"

## STEP 4. [최종] 자가 피드백 (Self-Feedback)

**Q. 고객이 우리 랜딩페이지를 보고 가질 첫 번째 생각은 무엇인가요?**

> ✅ **기대 반응**: "안전관리가 '의무'에서 '자발적 참여'로 바뀔 수 있다고? 게이미피케이션이라니 흥미롭네..."
> 
> 핵심 차별점 ("게이미피케이션", "직접 실행")을 직접적으로 언급하여
> 타겟 고객(현장 소장, 안전 담당자, 안전팀장)이 즉시 관심을 가질 수 있도록 설계

**Q. 경쟁사 대비 우리 페이지만의 '한 방(Kick)'은 무엇인가요?**

- [ ] `압도적인 권위(A)`
- [ ] `놀라운 시연 영상(B)`
- [x] `확실한 효율성 수치(C)` + **"게이미피케이션 + 자발적 참여"라는 명확한 차별점**

> 💡 **핵심 킥**: 단순한 안전관리 솔루션이 아닌,
> **"간접 실행에서 직접 실행으로"** + **"게이미피케이션을 통한 자발적 참여"**라는 메시지로
> 안전관리의 패러다임 전환을 제시

| B2B 특화 요소 | ✅ | 페르소나별 가치, 경영진 설득, SEO 키워드 |
| CTA 배치 | 4곳 | Hero(2개) + 하단 + 플로팅 |
| 디자인 톤 | 다크모드 + 전문적 | B2B SaaS 표준에 부합 |

### 핵심 차별화 포인트
1. **"직접 실행" 패러다임** - 간접 실행(수동적 관리)에서 직접 실행(자발적 참여)으로 전환
2. **게이미피케이션** - 안전관리에 게임 요소를 적용하여 참여 유도
3. **역할별 명확한 가치 제안** - 박소장/김대리/이팀장 페르소나 (핵심 Pain → 서비스 가치)
4. **AI 안전 지능** - "감이 아닌 기준 있는 안전관리" (위험도 분석, 태그 자동 제공)
5. **워크플로우 시각화** - 점검→승인→조치 흐름 자동화 및 기록
6. **증빙 체계** - 중대재해처벌법 대응 객관적 근거 확보
