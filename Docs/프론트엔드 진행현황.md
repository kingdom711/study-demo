# 프로젝트 진행 현황 보고서

> **프로젝트명**: 안전의 길 (Safety Quest Game)  
> **작성일**: 2024-12-20  
> **프로젝트 타입**: 게이미피케이션 기반 안전관리 플랫폼

---

## 📋 프로젝트 개요

**안전의 길**은 현장 안전관리를 자발적으로 모두가 참여하게 하는 게이미피케이션 플랫폼입니다. 중대재해처벌법 대응을 위한 법정 자율점검표 전산화, 역할 기반 워크플로우 자동화, AI 기반 위험도 분석 기능을 제공합니다.

### 핵심 가치 제안
- 법정 기준 체크리스트 전산화
- 역할 기반 워크플로우 자동화
- AI 기반 사진 위험도 분석
- "내 할 일" 중심 실무 UX

---

## 🛠 기술 스택

### 프론트엔드
- **프레임워크**: React 18.2.0
- **빌드 도구**: Vite 5.0.8
- **라우팅**: React Router DOM 6.20.0
- **스타일링**: 
  - Tailwind CSS 4.1.18 (최근 도입)
  - 커스텀 CSS (기존 스타일 유지)
- **언어**: JavaScript (ES6+)

### 백엔드 (예정)
- **프레임워크**: Spring Boot
- **AI 통합**: Google Gemini API (통합 준비 완료)

### 개발 환경
- **Node.js**: 18.x 이상
- **패키지 관리**: npm
- **개발 서버**: Vite Dev Server (포트 3000)
