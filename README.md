# 안전의 길 (Safety Quest) 🛡️

> **중대재해를 막는 안전관리, 이제 '간접 실행'에서 '직접 실행'으로**
>
> 게이미피케이션을 적용한 안전관리 솔루션

## 📋 프로젝트 개요

**안전의 길**은 현장 안전관리를 자발적으로 모두가 참여하게 하는 게이미피케이션 플랫폼입니다.

### 핵심 문제 (Problem)
- 법정 자율점검·안전서류가 **수기·분산 관리**되어 누락, 중복 발생
- 작업자-감독자-안전관리자 간 **워크플로우 단절**
- 사진은 있지만 **객관적 위험 판단 기준 없음**
- 사고 발생 시 **책임 추적 부담** (점검/승인/조치 여부 입증 어려움)

### 핵심 솔루션 (Solution)
1. **법정 기준 체크리스트 전산화** - 사다리/고소작업대/밀폐공간 표준 템플릿
2. **역할 기반 워크플로우 자동화** - 점검→승인→조치 전 과정 자동 기록
3. **AI 기반 사진 위험도 분석** - 안전/주의/위험 자동 판별
4. **"내 할 일" 중심 실무 UX** - 역할별 우선순위 업무만 노출

## ✨ 핵심 기능

### 🎯 체크리스트 시스템
- **법정 자율점검표 기반** 표준 템플릿
- **모바일 체크리스트** 작성 및 제출
- **자동 취합** 및 상태 관리

### 🔄 워크플로우 자동화
- **기술인**: 체크리스트 작성·사진 업로드
- **관리감독자**: 위험 항목 중심 승인/반려
- **안전관리자**: 전체 현장 위험현황 조회 및 조치 기록

### 🤖 AI 안전 지능
- **작업사진 위험도 분석** (안전/주의/위험)
- **위험 태그 자동 제공** (보호구 미착용, 추락 위험 등)
- **고위험 작업 상위 노출**

### � 증빙 체계
- 모든 점검·승인·조치 **타임스탬프 기록**
- 감사·점검 시 **즉시 제출 가능**
- **중대재해처벌법 대응** 객관적 근거 확보

## 🚀 시작하기

### 요구사항
- Node.js 18.x 이상
- npm 또는 yarn

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/your-repo/safety-quest-game.git

# 디렉토리 이동
cd safety-quest-game

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

개발 서버: `http://localhost:5173`
